# rule NO.1
我绝不杀死自己的时间。

# 到底时间是什么

竞争的本质。

想在竞争中取胜，时间是唯一纬度。

主导历史的因素，

短期（一年到几年）是政治，

中期（几年到几十年）是经济，

长期（几十年到几百年）则是技术。

# 紧迫感

一种帮你节省时间，
一种帮你消费时间。
因为有时间，所以才出现了管理。
