# 技术和专长

技术：人类利用工具和创造新工具的能力。
手艺：手艺首先是服务业。

我当初为什么选择技术，
其实是因为业务，
我选择了跟自己最有关的业务。
就是增加自己处理日常生活的能力。
这也就是软件最大的长处和魅力所在。

而现在呢，我因为市场选择了一个无法增长我能力的技术。

其实本质是兴趣爱好。
但是是你自主学习的东西，
有的学的比较好，或最近常用或最近比较热门。

计算机技术跟别的技术比如医学，法律，英语等。
还有本质的区别，
就在于计算机技术发展的速度要高很多，
换句话来说，就是现在学的东西变数很大。
所以计算机技术，并不需要太过熟练。
学不好也没关系。

你见过医学经常发生变化么。
难道说一个病现在得了以后就被消灭了？
计算机技术是个横向技术。
需要什么就会什么，是得有目标的，得寄存于另外一种技术之上
